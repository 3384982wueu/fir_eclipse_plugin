# Fir_Eclipse_Plugin

## 安装

> 		--> Help 
> 		--> Install New Software
> 		--> 填写上 http://121.201.63.30/
> 		--> 选中fir.im.category
> 		--> 安装弹出证书认证选择允许继续
> 		--> finish

## 插件位置
>       --> 安装完成之后重启
>       --> 工具栏就会有fir.im图标

## 使用
>       --> 单机fir.im图标会显示出对话框
>       --> 先填写[api_token](http://fir.im/apps)
>       --> 之后单机上传

## 提交反馈
> [使用 github issue 即可](https://github.com/FIRHQ/fir_eclipse_plugin/issues/new)
